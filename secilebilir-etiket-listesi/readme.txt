== Description ==

List all tags under the tag input box when you are writing a new post.

== Installation ==

It's too easy :)

1. Upload `secilebilir-etiket-listesi` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
